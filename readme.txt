Weapon name WEAPON_EXDUCK
Version: 1.0.0 
Creation date: 15-11-25

A new explosive rubberduck

**Install**
ensure npds_exduck in youre server.cfg

Add the weapon to ox_inventory
**Ox Inventory -> data -> weapons.lua** 

['WEAPON_EXDUCK'] = {
	label = 'Rubberduck',
	weight = 600,
	throwable = true,
},

Add the included inventory image to ox_inventory
Only tested on OX-INVENTORY
No support for other inventories
